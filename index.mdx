<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with Cowrie in minutes</p>
  
  <Steps>
    <Step title="Install the library">
      Choose your language and install Cowrie using your package manager:
      
      <CodeGroup>
      ```bash Go
      go get github.com/Neumenon/cowrie
      ```
      
      ```bash Python
      pip install cowrie
      ```
      
      ```bash Rust
      cargo add cowrie-codec
      ```
      
      ```bash TypeScript
      npm install cowrie
      ```
      </CodeGroup>
    </Step>
    
    <Step title="Import and encode data">
      Import the codec and encode your first message:
      
      <Tabs>
        <Tab title="Go">
          ```go
          import "github.com/Neumenon/cowrie/gen1"
          
          data, err := gen1.Encode(map[string]any{
              "name": "Alice",
              "age": 30,
          })
          ```
        </Tab>
        <Tab title="Python">
          ```python
          from cowrie import gen1
          
          data = gen1.encode({"name": "Alice", "age": 30})
          ```
        </Tab>
        <Tab title="Rust">
          ```rust
          use cowrie::gen1;
          
          let val = gen1::Value::Object(vec![
              ("name".to_string(), gen1::Value::String("Alice".to_string())),
          ]);
          let data = gen1::encode(&val)?;
          ```
        </Tab>
        <Tab title="TypeScript">
          ```typescript
          import { gen1 } from 'cowrie';
          
          const data = gen1.encode({ name: 'Alice', age: 30 });
          ```
        </Tab>
      </Tabs>
    </Step>
    
    <Step title="Decode and use">
      Decode the binary data back to your native data structures:
      
      ```go
      result, err := gen1.Decode(data)
      fmt.Printf("%v\n", result)
      ```
      
      <Note>Cowrie maintains wire format compatibility across all languages, so data encoded in Go can be decoded in Python, Rust, TypeScript, or C</Note>
    </Step>
  </Steps>
</div>
